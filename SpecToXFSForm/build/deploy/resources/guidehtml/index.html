Start Converting

1. Loading the Image

1. Open up the receipt specs (.pdf) and find the page with the receipt you want to convert.
2. Zoom in about 200% (doesn't need to be exact). The bigger the image of the receipt, the better.



3. Take a screenshot and crop a region around the receipt image (or use the snipping tool with "alt + s").







4. Make sure you have the full receipt image cropped out - including its borders but no other text or black pixels outside it!



Click to zoom in!
End result should look like this:

2. Gridding The Image

One you have your cropped image, you just need to load it into the app by going to File -> Open..

Assuming the image has been cropped correctly and doesn't contain any funky black pixels outside of the receipt image's border, click "Crop/Grid" and it should crop and grid it for you!


3. Defining Fields

Now just hit the "Define Fields" button and the GUI editor should open with your newly gridded image. To define a field just click on the starting point followed by the end point.

Once two red dots have been placed a pop up window will appear asking you to define the field name (mandatory), whether you want it to have an initial default value (labels/static fields need this) and its justification (receipt specs should tell you if they have to be centered or not).

Once your done hit "Save", or if you want that current GUI session to be cleared just hit "Clear Changes".

<\center>

Thats it! Just hit "Convert to XFS" and it will save your XFS form under the location posted to the Console.

4. Convert To XFS Form

Thats it! Just hit "Convert to XFS" and it will save your XFS form under the location posted to the Console.


<\center>

It's worth noting there is a "XFSFormSettings.xml" file which can be found under \app\resources\

Here you can edit things such as whether you want the receipt to use ROWCOLUMNS or MM, ROWCOLUMN-X and ROWCOLUMN-Y modifiers, Font and CPI (Characters Per Inch aka Font Size). The default should be suitable for the RBS project as of 07/04/2015.
